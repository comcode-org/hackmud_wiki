title: scripts.trust
---

scripts.trust lists the TRUST scripts.

### Security Level

FULLSEC

## Syntax

### CLI

```
scripts.trust
```

### Script

```
#fs.scripts.trust()
```

### Parameters

No known parameters.

### Return

Returns an array.

#### CLI

On the CLI, scripts.trust returns the array of strings formatted in 2 columns.

#### Script

{
