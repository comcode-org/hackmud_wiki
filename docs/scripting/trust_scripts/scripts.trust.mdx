title: scripts.trust
---

scripts.trust lists the TRUST scripts.

### Security Level

FULLSEC

## Syntax

### CLI

```
scripts.trust
```

### Script

```
#fs.scripts.trust()
```

### Parameters

No known parameters.

### Return

Returns an array.

#### CLI

On the CLI, scripts.trust returns the array of strings formatted in columns.

```
>>scripts.trust
accts.balance              corps.create               gui.vol                    scripts.quine              sys.upgrades
accts.balance_of_owner     corps.hire                 kernel.hardline            scripts.sys                sys.upgrades_of_owner
accts.transactions         corps.manage               market.browse              scripts.trust              sys.xfer_upgrade_to
accts.xfer_gc_to           corps.offers               market.buy                 scripts.user               sys.xfer_upgrade_to_caller
accts.xfer_gc_to_caller    corps.quit                 market.sell                sys.access_log             trust.me
autos.reset                corps.top                  market.stats               sys.breach                 users.active
chats.channels             escrow.charge              scripts.fullsec            sys.cull                   users.config
chats.create               escrow.confirm             scripts.get_access_level   sys.init                   users.inspect
chats.join                 escrow.stats               scripts.get_level          sys.loc                    users.last_action
chats.leave                gui.chats                  scripts.highsec            sys.manage                 users.top
```

#### Script

Called as a subscript, scripts.trust returns the array of strings in a single column by default.

## Example

```js

{
	return #fs.scripts.trust()
}
```
