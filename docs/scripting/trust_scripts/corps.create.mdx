---
title: corps.create
---

corps.create allows a user to create a player corporation for the cost of 1BGC.

### Security level

NULLSEC

## Syntax

### CLI

```
corps.create { name: "example_corp", confirm: true }
```

### Script

```
#ns.corps.create({ name: "example_corp", confirm: true })
```

### Parameters

#### name (required)

The 'name' argument specifies the name of the new player corporation.

#### confirm (required)

The 'confirm' argument confirms creation of new player corporation.

### Return

Returns an object.

#### CLI

```
>>corps.create { name: "example_corp", confirm: true }
Success

Transferred 1BGC to trust
```

#### Script

Same as CLI.

## Example

```
function(context, args)
{
	return #ns.corps.create({ name: "example_corp", confirm: true })
}

```
