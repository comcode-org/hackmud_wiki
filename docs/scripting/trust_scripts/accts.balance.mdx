---
title: accts.balance
description: "Returns the user's current GC balance"
---

((accts.balance)) returns the user's current ((%CGC%)) balance

### Security Level

HIGHSEC

## Syntax

### CLI

```
accts.balance
```

### Script

```
#hs.accts.balance()
```

### Parameters

No known parameters.

### Return

#### CLI

On the CLI ((accts.balance)) returns a ((%CGC%)) string

```
>>accts.balance
1M234K567GC
```

#### Script

In a script, #hs.accts.balance() returns a number

```
1234567
```

## Example

```js
function(context, args)
{
	let bal = #hs.accts.balance()
	return bal
}
```
