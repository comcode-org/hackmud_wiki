---
title: sys.loc
description: "Returns the calling system's loc"
---

((sys.loc)) displays the calling system's [[loc]].

### Security Level

LOWSEC

## Syntax

### CLI

```
sys.loc
```

### Script

```
#ls.sys.loc()
```

### Parameters

No known parameters.

### Return

Returns a string.

#### CLI

```
>>sys.loc
user.loc
```

#### Script

Same as CLI.

## Example

```js
function(context, args)
{
	return #ls.sys.loc()
}
```
