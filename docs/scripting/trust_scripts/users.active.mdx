---
title: users.active
description: "Displays the number of currently active users"
---

((users.active)) displays the number of currently active users.

### Security Level

FULLSEC

## Syntax

### CLI

```
users.active
```

### Script

```
#fs.users.active()
```

### Parameters

No known parameters.

### Return

Returns a number.

#### CLI

```
>>users.active
current users: 293

```

#### Script

```
293
```

## Example

```js
function(context, args)
{
	return #fs.users.active()
}
```
