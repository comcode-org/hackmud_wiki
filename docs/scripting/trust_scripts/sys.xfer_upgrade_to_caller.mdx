---
title: sys.xfer_upgrade_to_caller
---

accts.xfer_upgrade_to_caller allows a script to send upgrades to the caller of the script.

### Security Level

FULLSEC

## Syntax

### CLI

```
sys.xfer_upgrade_to_caller
```

### Script

Single upgrade:

```
#fs.sys.xfer_upgrade_to_caller({ to: "user", i: 0 })
```

Multiple upgrades:

```
#fs.sys.xfer_upgrade_to_caller({ to: "user", i: [ 0, 1, 2 ] })
```

### Parameters

#### i (required)

The 'i' argument takes a number or array of numbers and specifies which upgrades shall be sent.

### Return

Returns a Success or Failure object.

#### CLI

```
>>sys.xfer_upgrade_to_caller
Usage: (in subscript) #fs.sys.xfer_upgrade_to_caller({ i:<upgrade index or array>, sn:<upgrade SN or array>, memo:"<optional memo (50/1)>" })
```

## Example

```js
function(context, args)
{
	return #fs.sys.xfer_upgrade_to_caller({ i: 44 })
}
```
