---
title: gui.size
description: "Changes text size"
---

((gui.size)) specifies text size in the terminal window.

### Security Level

FULLSEC

## Syntax

### CLI

```
gui.size { i: 0 }
```

### Script

Cannot be called as a subscript.

### Parameters

#### i (required)

the '((%Ni%))' argument specifies the text size, as a number from ((%V-20%)) to ((%V20%)). The default value is set to ((%V0%)).

### Return

Returns a ((%LSuccess%)) or ((%DFailure%)) object.

#### CLI

No arguments or invalid arguments:

```
>>gui.size
Failure
Usage: gui.size { i:<-20 to 20> }
change your gui size settings. negative numbers reduce size positive increases.
current size: 0
```

Valid arguments:

```
>>gui.size { i: 0 }
Success
```

#### Script

Cannot be called as a subscript.
