---
title: corps.top
description: "Returns the top 10 player coporations by total GC"
---

((corps.top)) returns the domain's top 10 player corporations by total ((%CGC%)).

### Security level

NULLSEC

## Syntax

### CLI

```
corps.top
```

### Script

```
#ns.corps.top()
```

### Parameters

No known parameters.

### Return

Return an object.
