---
title: chats.tell
description: "Sends a private message to a specified user"
---

((chats.tell)) allows a user to send a private message to another user.

### Security level

FULLSEC

## Syntax

### CLI

```
chats.tell { to: "user", msg: "this is a private message" }
```

### Script

```
#fs.chats.tell({ to: "user", msg: "this is a private message" })
```

### Parameters

#### to (required)

The '((%Nto%))' argument specifies the user to send a message to.

#### msg (required)

The '((%Nmsg%))' argument specifies the private message to be sent.

### Return

Returns an object.

#### CLI

```
>>chats.tell { to: "user", msg: "this is a private message" }
Msg Sent

1555 to user :::this is a private message:::
```

#### Script

Same as CLI.

## Example

```js
function(context, args)
{
	return #fs.chats.tell({ to: "user", msg: "this is a private message" })
}
```
