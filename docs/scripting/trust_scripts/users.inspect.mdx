---
title: users.inspect
---

users.inspect displays the profile of a specified user.

### Security Level

HIGHSEC

## Syntax

### CLI

```
users.inspect { name: "seanmakesgames" }
```

### Script

```
#hs.users.inspect({ name: "seanmakesgames" })
```

### Parameters

#### name (required)

The 'name' argument takes a string and specifies the user to inspect.

### Return

Returns an object.
