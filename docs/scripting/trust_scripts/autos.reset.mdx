---
title: autos.reset
description: "Resets the caller's autocompletes"
---

((autos.reset)) clears out all of the caller's autocompletes.

### Security level

MIDSEC

## Syntax

### CLI

```
autos.reset
```

### Script

```
#ms.autos.reset()
```

### Parameters

No known parameters.

### Return

#### CLI

```
Success

-autocompletes reset-
```

#### Script

Same as CLI.

## Example

```js
function(context, args)
{
	return #ms.autos.reset()
}
```
