---
title: sys.status
description: "Returns the user's system status"
---

((sys.status)) returns the status of the user that runs it.

### Security Level

HIGHSEC

## Syntax

### CLI

```
sys.status
```

### Script

```
#hs.sys.status()
```

### Parameters

There are no known parameters.

### Return

### CLI

Returns a string.

```
>>sys.status
-system status nominal-
```

or

```
>>sys.status
-system is breached-
```

### Script

```
{
  hardline: 0,
  tutorial: "",
  breach: false
}
```

## Example

```js
function(context, args)
{
  return #hs.sys.status()
}
```
