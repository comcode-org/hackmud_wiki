---
title: sys.status
---

sys.status returns the status of the user that runs it.

### Security Level

HIGHSEC

## Syntax

### CLI

```
sys.status
```

### Script

```
#fs.sys.status()
```

### Parameters

There are no known parameters.

### Return

sys.specs returns a string.

### CLI

```
>>sys.status
-system status nominal-
```

or

```
>>sys.status
-system is breached-
```

### Script

```
{
  hardline: 0,
  tutorial: "",
  breach: false
}
```

## Example

```js
function(context, args)
{
  return #hs.sys.status()
}
```
