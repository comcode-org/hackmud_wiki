---
title: corps.hire
description: "Hires another user to a player corporation"
---

((corps.hire)) allows a player corporation owner or admin to hire another user.

### Security level

NULLSEC

## Syntax

### CLI

```
corps.hire { name: "example_user" }
```

### Script

```
#ns.corps.hire()
```

### Parameters

#### name (required)

The '((%Nname%))' argument specifies the user to be hired.

### Return

Returns an object.

#### CLI

```
>>corps.hire{name: "example_user"}
Success

```

#### Script

Same as CLI.

## Example

```js
function(context, args)
{
	return #ns.corps.hire({ name: "example_user" })
}
```
