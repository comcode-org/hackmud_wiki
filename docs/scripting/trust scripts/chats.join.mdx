---
title: chats.join
---

## Description

chats.join allows a user to join a specified channel.

## Syntax

On the Command Line Interface:

```
chats.join { channel:"example_channel" }
```

As a subscript:

```
#ms.chats.join({ channel:"example_channel" })
```

### Parameters

#### channel (required)

Specifies which channel to join.

### Return

#### CLI

```
>>chats.join { channel: "example_channel" }
Success

1555 example_channel <user> :::user joined channel:::
```

#### Script

Same as CLI.

## Example

```
function(context, args)
{
	return #ms.chats.join({ channel: "example_channel" })
}

```
