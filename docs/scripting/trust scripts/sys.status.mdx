---
title: sys.status
---

## Description

sys.status returns the status of the user that runs it.

## Syntax

On the Command Line Interface:

```
sys.status
```

As a subscript call:

```
#fs.sys.status()
```

### Parameters

There are no known parameters.

### Return

In the main terminal window, sys.status returns one of the following:

```
>>sys.status
-system status nominal-
```

or

```
>>sys.status
-system is breached-
```
