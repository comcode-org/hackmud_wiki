---
title: sys.status
---

## Description

sys.status returns the status of the user that runs it.

## Syntax

### CLI

```
sys.status
```

### Script

```
#fs.sys.status()
```

### Parameters

There are no known parameters.

### Return

sys.specs returns a `String`

### CLI

```
>>sys.status
-system status nominal-
```

or

```
>>sys.status
-system is breached-
```

### Script

```
{
  hardline: 0,
  tutorial: "",
  breach: false
}
```
