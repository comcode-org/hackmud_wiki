---
title: chats.channels
---

## Description

chats.channels returns all the channels a user is in currently.

## Syntax

On the Command Line Interface:

```
chats.channels
```

As a subscript:

```
#ms.chats.channels()
```

### Parameters

No known parameters.

### Return

#### CLI

```
>>chats.channels
0000
0001
example_channel
```

#### Script

Same as CLI

## Example

```
function(context, args)
{
	return #ms.chats.channels()
}

```
