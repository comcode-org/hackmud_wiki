---
title: corps.quit
---

## Description

corps.quit allows a player corporation member remove themselves from the player corporation.

## Syntax

### CLI

```
corps.quit { confirm: true }
```

As a subscript:

```
#ns.corps.quit({ confirm: true })
```

### Parameters

#### confirm (required)

Confirms quitting the player corporation.

### Return

#### CLI

```
>>corps.quit { confirm: true }
Success
```

#### Script

Same as CLI.

## Example

```
function (context,args)
{
  return #ns.corps.quit({ confirm: true })
}
```
