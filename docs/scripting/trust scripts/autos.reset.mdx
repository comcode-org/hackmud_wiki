---
title: autos.reset
---

## Description

autos.reset clears out all of a users autocompletes

## Syntax

On the Command Line Interface:

```
autos.reset
```

As a subscript:

```
#ms.autos.reset()
```

### Parameters

No known parameters.

### Return

#### CLI

```
Success

-autocompletes reset-
```

#### Script

Same as CLI.

## Example

```
function(context, args)
{
	return #ms.autos.reset()
}

```
