---
title: chats.leave
---

## Description

chats.leave allows a user to leave a specified channel.

## Syntax

### CLI

```
chats.leave { channel: "example_channel" }
```

As a subscript:

```
#ms.chats.leave({ channel: "example_channel" })
```

### Parameters

#### channel (required)

### Return

#### CLI

```
>>chats.leave {channel:"example_channel"}
Success

1613 example_channel <user> :::user left channel:::
```

#### Script

Same as CLI.

## Example

```
function(context, args)
{
	return #ms.chats.leave({ channel: "example_channel" })
}

```
