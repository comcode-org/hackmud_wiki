---
title: hackmud color codes
---

It is possible to color hackmud text using built-in color codes.

## Syntax

Color codes are denoted by a pair of backticks (`) enclosing the text and an alphanumeric color identifier.
The identifier is placed before the text, inside the backticks.

An example:

```
\`<ID>text\`
```
