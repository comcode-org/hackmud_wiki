---
title: scripts.lib().get_log()
---

## Description

Returns the logged strings from log().

## Syntax

```js
return #fs.scripts.lib().get_log()
```

### Parameters

There are no known parameters.

### Return

Returns an array.

```js
"hello world"
```

## Example

```JS
function(context,args){
  let l = #fs.scripts.lib()
  l.log('hello world')
	return l.get_log()
}
```
