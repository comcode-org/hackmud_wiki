---
title: scripts.lib().clone()
---

A function that checks if the input object is defined, and if so, clones the object and returns the result of the operation. If the object is not defined, clone will return "undefined".

## Syntax

```js
#fs.scripts.lib().clone(i);
```

Where "i" is the object to be cloned.

### Parameters

# i

The input to be evaluated.

### Return

Returns the structure of the input object.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_obj = {
    property: "value"
  };

  return l.clone(my_obj);
}
```
