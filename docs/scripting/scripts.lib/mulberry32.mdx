---
title: scripts.lib().mulberry32()
---

## Description

A function that gets a random number from a numerical seed based on the mulberry32 RNG.

## Syntax

```js
#fs.scripts.lib().mulbery32(1337)
```

### Parameters

Takes a number as its parameter.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_number = 1337
  return l.mulberry32(my_number)
}
```
