---
title: .mulberry32()
---

A function that returns a PRNG function from a numerical seed based on the mulberry32 algorithm.

## Syntax

```js
#fs.scripts.lib().mulberry32(num);
```

### Parameters

#### num

The number to serve as the seed for RNG.

### Return

Returns a function.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_number = 1337;
  const rng = l.mulberry32(my_number);

  return rng();
}
```
