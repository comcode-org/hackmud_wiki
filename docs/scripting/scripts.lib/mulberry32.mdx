title: scripts.lib().mulberry32()
---

## Description

A function that takes a number and turns it into a hash code based on the mulberry32 algorithm.

## Syntax

```js
#fs.scripts.lib().mulbery32(1337)
```

### Parameters

No known parameters.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_number = 1337
  return l.mulberry32(my_number)
}
```
