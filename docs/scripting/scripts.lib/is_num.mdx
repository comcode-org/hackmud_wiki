---
title: .is_num()
---

A function to test whether the input is a number.

## Syntax

```js
#fs.scripts.lib().is_num(i);
```

### Parameters

#### i

"i" is the input to be tested.

### Return

Returns a boolean.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_test_input = 1337;

  return l.is_num(my_test_input);
}
```
