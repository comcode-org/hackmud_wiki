---
title: .to_game_timestr()
---

Takes a JavaScript `new Date()` object and converts it into a time string in the `YYMMDD.HHSS` format.

## Syntax

```js
#fs.scripts.lib().to_game_timestr(date);
```

### Parameters

#### date

Takes a JavaScript `new Date()` object as its parameter.

### Return

Returns a string.

```
240227.0044
```

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const current_time_and_date = new Date();

  return l.to_game_timestr(current_time_and_date);
}
```
