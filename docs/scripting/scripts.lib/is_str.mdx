---
title: scripts.lib().is_str()
---

A function to test whether input is a string.

## Syntax

```js
#fs.scripts.lib().is_str(i);
```

Where "i" is the input to be tested.

### Parameters

No known parameters.

### Return

Returns a boolean.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_test_input = "I am a string.";

  return l.is_str(my_test_input);
}
```
