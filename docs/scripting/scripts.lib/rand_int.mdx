---
title: scripts.lib().rand_int()
---

## Description

A function that returns a random number between two parameters.

## Syntax

```js
scripts.lib().rand_int(1, 4);
```

### Parameters

#### Min

The minimum number to generate. (inclusive)

#### Max

A number which is one greater than the maximum number to generate. (exclusive)

### Return

Returns a random number between the min and max:

```
3
```

## Example

```js
function(context, args) {
  var l = #fs.scripts.lib();
  return l.rand_int(1, 4);
}
```
