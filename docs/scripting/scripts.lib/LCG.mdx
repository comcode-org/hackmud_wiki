---
title: scripts.lib().LCG()
---

## Description

Takes a numerical seed and generates a random number based on the LCG RNG.

## Syntax

```js
#fs.scripts.lib().LCG(12345)
```

### Parameters

Takes a number as its parameter.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  my_seed = 12345
  return l.LCG(my_seed)
}
```
