---
title: .LCG()
---

Takes a numerical seed and generates a random number based on the LCG RNG.

## Syntax

```js
#fs.scripts.lib().LCG(seed)
```

### Parameters

#### seed

The number to serve as the seed for RNG.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  my_seed = 12345;

  return l.LCG(my_seed);
}
```
