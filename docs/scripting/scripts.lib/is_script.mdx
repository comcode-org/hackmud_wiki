---
title: scripts.lib().is_script()
---

## Description

A function that checks if the input is a valid script.

## Syntax

```js
#fs.scripts.lib().is_script(scriptname)
```

### Parameters

# scriptname

The string representation of a scriptname, e.g. "scripts.trust"

### Return

Returns a string

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const scriptname = "scripts.trust"
  return l.is_script(scriptname)
}
```
