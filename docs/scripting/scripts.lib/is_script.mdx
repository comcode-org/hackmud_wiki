---
title: .is_script()
---

A function that checks if a script is being ran as a subscript, given context and args parameters. Useful for differentiating behavior in a script between calls on the CLI vs in subscripts

## Syntax

```js
#fs.scripts.lib().is_script(context, args);
```

### Parameters

# scriptname

The string representation of a scriptname, e.g. "scripts.trust"

### Return

Returns a boolean.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const scriptname = "scripts.trust";

  return l.is_script(context, arg) ? "this was called with is_script or was called as a subscript" : "etc";
}
```
