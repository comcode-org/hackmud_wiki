---
title: .hash_code()
---

A function which creates a hash code out of an input string.

## Syntax

```js
#fs.scripts.lib().hash_code(str)
```

### Parameters

#### str

The string to be hashed.

### Return

Returns a string.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_string;

  return l.hash_code(my_string);
}
```
