---
title: scripts.lib().get_user_from_script()
---

A function that extracts the user name of a script owner based on the script name.

## Syntax

```js
#fs.scripts.lib().get_user_from_script(scriptname);
```

### Parameters

# scriptname

The string representation of a script name, e.g. "scripts.trust"

### Return

Returns a string.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const scriptname = "scripts.trust";

  return l.get_user_from_script(scriptname);
}
```
