---
title: scripts.lib().is_bool()
---

## Description

A function to test whether the input is a boolean.

## Syntax

```js
#fs.scripts.lib().is_bool(i)
```

Where "i" is the input to be tested.

### Parameters

No known parameters.

### Return

Returns a boolean.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_test_input = true
  return l.is_bool(my_test_input)
}
```
