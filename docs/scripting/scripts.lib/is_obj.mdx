---
title: .is_obj()
---

A function to test whether the input is an object.

## Syntax

```js
#fs.scripts.lib().is_obj(i);
```

### Parameters

#### i

"i" is the input to be tested.

### Return

Returns a boolean.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_test_input = { i_am: "an object" };

  return l.is_obj(my_test_input);
}
```
