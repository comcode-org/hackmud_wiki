---
title: scripts.lib().lerp()
---

### Description

LERP stands for Linear Interpolation; a mathematical function in which a value interpolates between 'start' and 'end' values.
Formula: start + (end - start) \* t

## Syntax

```js
return #fs.scripts.lib().lerp(5, 0, 10) // Result will be 10
```

### Parameters

#### Start

The starting value.

#### End

The ending value.

#### t

The interpolation factor, usually between 0 and 1. If t is 0 the result is equal to Start. When the value is 1, the result is equal to End. Values between 0 and 1 will interpolate between Start and End.

### Return

Returns a number.

```
10
```

## Example

```js
function ( context, args ) {
  let l = #fs.scripts.lib();
  return l.lerp( 5, 0, 10 );
}
```
