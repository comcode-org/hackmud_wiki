## A function that checks if a specified date is not today's date.

## Syntax

```js
#fs.scripts.lib().is_not_today(date);
```

### Parameters

# date

A JavaScript `Date` object.

### Return

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const date = new Date();

  return l.is_not_today(date);
}
```
