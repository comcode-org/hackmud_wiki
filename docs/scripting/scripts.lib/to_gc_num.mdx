---
title: scripts.lib().to_gc_num()
---

## Description

Takes a GC string and turns it into a number.

## Syntax

```js
#fs.scripts.lib().to_gc_num("100M200K300GC")
```

### Parameters

Takes a GC string as its parameter.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_gc_string = "100M200K300GC"
  return l.to_gc_num(my_gc_string)
}
```
