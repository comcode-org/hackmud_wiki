---
title: scripts.lib().xoshiro128ss()
---

A function that gets a random number from 4 numerical seeds based on the xoshiro128ss RNG.

## Syntax

```js
#fs.scripts.lib().xoshiro128ss(a, b, c, d);
```

### Parameters

#### a

First seed number.

#### b

Second seed number.

#### c

Third seed number.

#### d

Fourth seed number.

### Return

Returns a number.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const seed_1 = 35484;
  const seed_2 = 97456;
  const seed_3 = 35241;
  const seed_4 = 85746;

  return l.xoshiro128ss(seed_1, seed_2, seed_3, seed_4);
}
```
