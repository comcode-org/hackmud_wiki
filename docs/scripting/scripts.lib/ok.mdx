---
title: .ok()
---

Helper function that creates a formatted ((%LSuccess%)) object.

## Syntax

```js
return scripts.lib().ok();
```

### Parameters

No known parameters.

### Return

## CLI

```
Success
```

## Script

```
{
  ok: true
}
```

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  return l.ok();
}
```
