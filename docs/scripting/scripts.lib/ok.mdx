---
title: scripts.lib().ok()
---

## Description

Helper function that creates a formatted Success object.

## Syntax

```js
return scripts.lib().ok()
```

### Parameters

No known parameters.

### Return

```js
{ok:true}
```

## Example

```js
function( context, args ) {
    var l = #fs.scripts.lib()
    return l.ok()
}
```
