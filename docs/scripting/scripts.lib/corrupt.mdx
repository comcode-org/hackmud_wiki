---
title: .corrupt()
---

Applies corruption throughout a given string, in the specified frequency.

## Syntax

```js
#fs.scripts.lib().corrupt(str, corruption);
```

### Parameters

#### str

The string to be corrupted.

#### corruption

An integer specifying the frequency at which to corrupt the string. The actual frequency is given by using this integer as an index into [[.corruption_chars]]. Invalid indexes result in no corruption.

### Return

Returns a string.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  const my_string = "I am a string to be corrupted.";

  return l.corrupt(my_string, 3);
}
```
