---
title: scripts.lib().get_date()
---

## A safe internal `new Date()` invocation.

## Syntax

```js
#fs.scripts.lib().get_date();
```

### Parameters

Same as JavaScript's `new Date()`.

### Return

Returns a `Date` object.

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();

  return l.get_date();
}
```
