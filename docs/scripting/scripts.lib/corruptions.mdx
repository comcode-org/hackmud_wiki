---
title: .corruptions
---

The common corruption frequencies used in-game.

## Syntax

```js
#fs.scripts.lib().corruptions;
```

### Value

This property is the literal array:  
`[0, 1, 1.5, 2.5, 5]`

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();

  return l.corruptions;
}
```
