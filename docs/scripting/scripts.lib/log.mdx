---
title: .log()
---

Pushes strings into an array that can be returned later for logging/debugging.

## Syntax

```js
#fs.scripts.lib().log(logging_data);
```

### Parameters

#### logging_data

Data you wish to log to the logging array. Can be of any data type.

### Return

Does not return. In order to return the logged array, use [[.get_log()]]

## Example

```js
function(context, args) {
  const l = #fs.scripts.lib();
  l.log("hello world");
  return l.get_log();
}
```
