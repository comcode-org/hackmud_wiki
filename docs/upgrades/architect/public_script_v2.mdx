> public_script_slot allows you to mark scripts as public.

public_script_v2 is a tier 2 upgrade. It spawns at ((%0noob%)), ((%1kiddie%)), ((%2h4x0r%)) and ((%3h4rdc0r3%)) rarities.

## Stats

### slots

Displays the number of additional scripts a user can make public.
