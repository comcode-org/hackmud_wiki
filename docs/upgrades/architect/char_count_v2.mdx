> char_count adds space for additional characters in every script.

char_count_v2 is a tier 2 upgrade. It spawns at ((%0noob%)), ((%1kiddie%)), ((%2h4x0r%)) and ((%3h4rdc0r3%)) rarities.

## Stats

### chars

Displays the number of characters this upgrade adds.

## Behavior

Loading this upgrade increases the number of characters available for writing scripts.
