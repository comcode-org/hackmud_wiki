> CON_TELL attempts to send a tell to the owner of the lock

CON_TELL is a tier 1 [[lock]] created by the CONTEXT corporation.

## Stats

CON_TELL has no additional stats. The upgrade only spawns at ((%1kiddie%)) rarity.

## Behavior

CON_TELL does not take in any user input, and will be immediately activated upon encountering it. The lock sends a chats.tell message to the targeted user, then allows the attacker to proceed to the next lock. Every subsequent call will send another chats.tell message to the targeted user.

## Solving

Solving is not required for this lock.
