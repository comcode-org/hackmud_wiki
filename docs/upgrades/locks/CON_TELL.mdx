> CON_TELL attempts to send a tell to the owner of the lock

CON_TELL is a tier 1 [[lock]].

## Stats

CON_TELL has no additional stats. The upgrade only spawns at 'kiddie' rarity.

## Behavior

CON_TELL has no user input available. The lock is triggered on a breach attempt and sends chats.tells to the targeted user.

## Solving

Solving is not required for this lock.
