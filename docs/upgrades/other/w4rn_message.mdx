> sys.w4rn_message allows you to change the message in Archaic Labs' w4rn upgrade.

w4rn_message is a tier 1 upgrade. It spawns at ((%0noob%)), ((%1kiddie%)) and ((%2h4x0r%)) rarities.

## Behavior

When a w4rn_message is loaded on a user's system, sys.w4rn_message can be executed to change the message of the [[w4rn]] lock.

Syntax:

```
Usage: sys.w4rn_message { msg: "<message (100)>" }
```
