> channel_count allows you to connect to more chat channels.

channel_count_v1 is a tier 1 upgrade. Spawns at 'noob', 'kiddie', and 'h4x0r' rarities.

## Stats

### count

The 'count' stat displays how many additional channels a user can join when this upgrade is equipped.

## Behavior

channel_count_v1 increases the number of chat channels a user can join. A user is able to join 2 channels by default.

```
>>chats.join {channel:"6667"}
Success

>>chats.join {channel:"6668"}
Success

>>chats.join {channel:"6669"}
Failure
you cannot join any more channels
```
